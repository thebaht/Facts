# Facts

## 1+2+3+4+5+... = ⁻¹⁄₁₂

- S₁ &nbsp;&nbsp;=&thinsp; 1 - 1 + 1 - 1 + 1 - 1 + 1 - ... &nbsp;&nbsp;=&thinsp; ½  &emsp;&emsp;[Grandi's series](https://en.wikipedia.org/wiki/Grandi%27s_series)
- S₂ &nbsp;&nbsp;=&thinsp; 1 - 2 + 3 - 4 + 5 - 6 + 7 - ... &nbsp;&nbsp;=&thinsp; ¼  
&thinsp;&nbsp;&nbsp;&nbsp;&nbsp; 2 • S₂ &nbsp;&nbsp;=&thinsp;  1 - 2 + 3 &thinsp;- 4 + 5 &thinsp;- 6 + 7 &thinsp;- &thinsp;...  
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; + [ 1 - &thinsp;2 + 3 - &thinsp;4 + 5 - &thinsp;6 + ... &thinsp;]  
⇒ &nbsp; 2 • S₂ &nbsp;&nbsp;=&thinsp;  1 + ( -2+1 ) + ( 3-2 ) + ( -4+3 ) + ( 5-4 ) + ( -6+5 ) + ( 7-6 ) - ...  
⇒ &nbsp; 2 • S₂ &nbsp;&nbsp;=&thinsp; 1 - 1 + 1 - 1 + 1 - 1 + 1 - ...  
⇒ &nbsp; 2 • S₂ &nbsp;&nbsp;=&thinsp; ½  
⇒ &nbsp; S₂ &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&thinsp;=&thinsp; ¼  

- S &nbsp;&nbsp;=&thinsp; 1 + 2 + 3 + 4 + 5 + 6 + 7 + ... &nbsp;&nbsp;=&thinsp; - ¹⁄₁₂  
&thinsp;&nbsp;&nbsp;&nbsp;&nbsp; S - S₂ &nbsp;&nbsp;=&thinsp; 1 + 2 + 3 + 4 + 5 + 6 + 7 + ...  
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; - [ 1 &thinsp;- &thinsp;2 + 3 &thinsp;- 4 + 5 &thinsp;- 6 &thinsp;+ 7 &thinsp;- ... &thinsp;]  
⇒ &nbsp; S - S₂ &nbsp;&nbsp;=&thinsp; ( 1&thinsp;-&thinsp;1 ) + ( 2&thinsp;-&thinsp;(-2) ) + ( 3&thinsp;-&thinsp;3 ) + ( 4&thinsp;-&thinsp;(-4) ) + ( 5&thinsp;-&thinsp;5 ) + ( 6&thinsp;-&thinsp;(-6) )  
⇒ &nbsp;  S - S₂ &nbsp;&nbsp;=&thinsp; 4 + 8 + 12 + ...  
⇒ &nbsp;  S - S₂ &nbsp;&nbsp;=&thinsp; 4 • (1+2+3+4+5+...)  
⇒ &nbsp;  S - ¼ &nbsp;&nbsp;=&thinsp; 4 • S  
⇒ &nbsp;  &nbsp;&nbsp; - ¼ &nbsp;&nbsp;=&thinsp; 3 • S  
⇒ &nbsp; &nbsp;&nbsp; - ¹⁄₁₂ &nbsp;=&thinsp; S  

## Hair to proton size comparison

Protons are what scientist are colliding at CERN, and the channel they are passing them through is the width of a hair. But how big is that compared to the protons?  

- If a hair had the diameter of the earth,  
⇒ &nbsp;  the size of a cell would be the distance from paris to rome  
⇒ &nbsp;  a protein inside the cell would be ~6 soccer fields across  
⇒ &nbsp;  an atom inside the protein would be the length of a bus  
⇒ &nbsp;  the nucleus of the atom would be a grain of rice  
⇒ &nbsp;  a proton would be a grain of salt

So it's kinda insane we can line them up and make them hit each other on purpose.
